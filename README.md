# YLEON

Catalogo digital para curadoria de mobiliario, acabamentos e projetos de arquitetura.

## Desenvolvimento

Requisitos:

- Node.js
- npm

Instale as dependencias e inicie o servidor local:

```sh
npm install
npm run dev
```

## Scripts

```sh
npm run build
npm run lint
npm test
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
